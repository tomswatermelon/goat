# Goat

- Gulp
- Sass
- Vue 
- BrowserSync 

## Usage

### Start using
Run `npm install`.

### Development
Run `gulp dev`, browser will load the `dist/` folder and auto refresh upon changes.

Run `gulp img`, to pipe all images from `src/img/` to `dist/img/`.

### Production
Run `gulp production`, all css and js will be complied and compressed.
